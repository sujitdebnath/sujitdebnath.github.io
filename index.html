Research Works
  • Stroke Risk Prediction

    Sujit Debnath, Md. Asif Khan, Kazi Asfaq Ahmed Ador

    Several Machine Learning techniques such as Logistic Regression, Support Vector Classifier, etc along with Synthetic Minority Over-sampling Technique (SMOTE) has been used to predict the risk of stroke on 16 different features excluding the class.

  • Bengali Short Speech Commands Recognition

    Shakil Ahmed Sumon, Joydip Chowdhury, Sujit Debnath

    A robust dataset has been collected with 50 classes which is first of its kind in the context of Bangladesh. Purpose of the project was to classify Bengali short speech command using Convolutional Neural Network (CNN).
